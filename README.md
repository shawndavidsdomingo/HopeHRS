# HopeHRM — Hope, Inc. Human Resources Management System

A web-based HRM system built with Vite + React 18, Tailwind CSS, and Supabase.

---

## 🛠️ Tech Stack
- **Frontend**: Vite + React 18
- **Styling**: Tailwind CSS v3
- **Backend / DB**: Supabase (PostgreSQL)
- **Auth**: Supabase Auth (Google OAuth only — no passwords)
- **Routing**: React Router v6
